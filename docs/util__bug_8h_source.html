51 /* Non-fatal bug assertions. The "unreached" variants mean "this line should
52  * never be reached." The "once" variants mean "Don't log a warning more than
53  * once".
54  *
55  * The 'BUG' macro checks a boolean condition and logs an error message if it
56  * is true. Example usage:
57  * if (BUG(x == NULL))
58  * return -1;
59  */
60
